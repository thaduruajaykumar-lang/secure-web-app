# Secure Web Application

A Flask-based secure web application implementing:

## Features
- User Registration
- Secure Login
- Password Hashing
- Session Management
- SQLite Database

## Technologies Used
- Python
- Flask
- SQLAlchemy
- SQLite
- HTML
- CSS

## Installation

pip install -r requirements.txt

## Run

python app.py
